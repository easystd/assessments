# easystd/assessments – Syphilis Professional Assessment

Enterprise-grade anonymous symptom checker built with Next.js 15 + Tailwind CSS.

## Features
- Professional medical UI (shadcn-style)
- 6-step guided assessment with conditional logic
- Dynamic risk scoring (Low / Moderate / High)
- Strong medical disclaimers on every screen
- Fully responsive & mobile-first
- Zero data collection / anonymous by design
